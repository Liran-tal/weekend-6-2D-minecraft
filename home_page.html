Description

Each tool can be used to extract certain materials as specified below.
You can collect materials and store them in an inventory until it is full.
Use the stored materials to build buildings or natural structures.
* Note: In order to obtain new materials the inventory must have unoccupied slots.

The purpose of the game is for me to practice the combination of Javascript, HTML and CSS and for you to QA my exercise :D

Tool Uses

  • Pickaxe: Boulders
  • Shovel: Soil
  • Axe: Trees
